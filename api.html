API Reference#

This page contains the full API documentation for py2Dmol.

Main Module#

A Python library for visualizing protein structures in 2D.

class py2dmol.View(size=(500, 500), color='auto', *, shadow=True, outline=True, width=3.0, rotate=False)[source]#

Bases: object

A class for visualizing protein structures in 2D.

__init__(size=(500, 500), color='auto', *, shadow=True, outline=True, width=3.0, rotate=False)[source]#

Initialize a protein structure viewer.

Parameters:

  • size (tuple[int, int]) – Width and height of the viewer in pixels. Defaults to (500, 500).

  • color (Literal['auto', 'rainbow', 'chain']) – Color mode - “auto” (chooses based on chains), “rainbow”, or “chain”.

  • shadow (bool) – Whether to enable shadow effect. Defaults to True.

  • outline (bool) – Whether to enable outline effect. Defaults to True.

  • width (float) – Line width for rendering. Defaults to 3.0.

  • rotate (bool) – Whether to enable rotation. Defaults to False.

add(coords, plddts=None, chains=None, atom_types=None, *, new_traj=False, trajectory_name=None)[source]#

Add a new frame of data to the viewer.

If this is the first time ‘add’ is called, it will display the viewer.

Parameters:

  • coords (ndarray) – Nx3 array of coordinates.

  • plddts (ndarray | None) – N-length array of pLDDT scores.

  • chains (list[str] | None) – N-length list of chain identifiers.

  • atom_types (list[str] | None) – N-length list of atom types (‘P’, ‘D’, ‘R’, ‘L’).

  • new_traj (bool) – If True, starts a new trajectory. Defaults to False.

  • trajectory_name (str | None) – Name for the new trajectory. Only used if new_traj=True.

Return type:

None

add_pdb(filepath, chains=None, *, new_traj=False, trajectory_name=None)[source]#

Load a structure from a PDB or CIF file and add it to the viewer.

Multi-model files are added as a single trajectory.

Parameters:

  • filepath (str) – Path to the PDB or CIF file.

  • chains (list[str] | None) – Specific chains to load. Defaults to all.

  • new_traj (bool) – If True, starts a new trajectory. Defaults to False.

  • trajectory_name (str | None) – Name for the new trajectory. Only used if new_traj=True.

Return type:

None

Constants#

py2dmol.viewer.IS_COLAB#

bool(x) -> bool

Returns True when the argument x is true, False otherwise. The builtins True and False are the only two instances of the class bool. The class bool is a subclass of the class int, and cannot be subclassed.

Boolean flag indicating whether the code is running in Google Colab.
